# Linked List

Implement a Singly Linked List Data Structure

## Challenge
Implement a Node Class (with Next and Value properties), a LinkedList Class (With Head and Current Properties) as well as:
- Constructor to require one Node
- Add (O(1))
- Print (O(n))
- Append(O(n))
- AddBefore(O(n))
- AddAfter(O(n))
