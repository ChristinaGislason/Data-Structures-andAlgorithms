# Linked List: Reverse

Reverse a Linked List.

## Challenge

To write a function to reverse a Singly Linked List.

## Solution

Input:	head->[3]->[2]->[1]	
Output: head->[1]->[2]->[3]

Output
head->['a']->['b']->['c']	head->['c']->['b']->['a']
