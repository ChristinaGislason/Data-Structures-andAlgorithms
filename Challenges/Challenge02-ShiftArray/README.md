# Shift Array
Insert and shift an array in middle at index

## Challenge
A function called insertShiftArray which takes in an array and the value to be added. 
Returns an array with the new value added at the middle index.

## Solution
Example: [2,4,6,8], 5
Output: [2,4,5,6,8]
